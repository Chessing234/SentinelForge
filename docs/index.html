Cyber training platform

Train defenders. Place talent. Bill orgs.

SentinelForge is an open-source platform for hands-on security labs, live training sessions, hiring workflows, Slack integrations, and Stripe-backed organization billing.

What you get

Hands-on labs

Scenario browser, training sessions, and Socket.IO-powered live environments.

Workforce

Jobs, applications, placement matching, and progress tracking.

Enterprise

Admin metrics, org management, and Stripe checkout and customer portal.

Integrations

Slack OAuth and events, optional Google sign-in, Gemini-powered mentor.

About this site

This page is static (hosted on GitHub Pages). The full SentinelForge product runs as a Node.js server with PostgreSQL, Redis, and WebSockets — deploy it with Docker on a VM or container host. See the repository README for docker-compose.prod.yml and environment setup.
